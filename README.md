 <h1>📚 E-Saraswati - School Management Institute </h1> 
 <p>
   E-Saraswati is a modern School Management System designed to digitize and simplify academic and administrative operations.
The platform integrates biometric attendance, geo-fencing technology, and centralized management modules to improve efficiency, transparency, and security in educational institutions.
 </p>

 <h2>🚀 Features </h2>
 <ul>
   <h3>Biometric Teacher Attendence </h3>
   <li>Secure attendance system using biometric authetication . </li>
   <li>Ensures accurate teacher presence tracking. </li>

   <h3>Geo-fencing Integration </h3>
   <li>Attendence can only be marked within the school premises.  </li>
   <li>Prevents remote or unathorized check-ins. </li>

   <h3>Student Management </h3>
   <li>Student admission and profile management  </li>
   <li>Track academic records and details. </li>

   <h3>Attendance Management </h3>
   <li>Daily attendence for teachers and students.  </li>
   <li>Real-time monitoring and reports. </li>

   <h3>Administration Management </h3>
   <li>Centralized dashboard for managing school activities.  </li>
   <li>View analytics and reports. </li>

   <h3>Secure Authentication </h3>
   <li>Role-based login system for administrators and staff.  </li>
  
 </ul>
